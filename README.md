# Din Din

This is a application based on [Michael Hartl's Rails Tutorial](https://www.railstutorial.org/). It's designed to store
and serve dinner ideas, pun intended.

## Requirements
- Ruby 2.5
- Rails 5.2

## Getting Started

First clone the repo and install the needed gems.

```
bundle
```
